Role-aware API quick reference

All routes under /api/** require bearer auth except POST /api/auth/token.

Role key

USER Read APIs for alerts, criteria, and weather data.

ADMIN Includes USER access plus criteria writes, pending alerts, and expire alert.

Method Path Role Purpose
POST /api/auth/token Public Issue JWT access token
POST /api/criteria ADMIN Create alert criteria
PUT /api/criteria/{criteriaId} ADMIN Update alert criteria
DELETE /api/criteria/{criteriaId} ADMIN Delete alert criteria
GET /api/criteria/user/{userId} USER Read user criteria (supports optional filters)
GET /api/criteria/{criteriaId} USER Read one criteria record
GET /api/alerts/user/{userId} USER Get user alert feed
GET /api/alerts/{alertId} USER Get alert details by ID
GET /api/alerts/criteria/{criteriaId}/history USER Get alert history for one criteria
GET /api/alerts/pending ADMIN Get all pending alerts
POST /api/alerts/{alertId}/acknowledge USER Acknowledge an alert
POST /api/alerts/{alertId}/expire ADMIN Expire an alert
GET /api/weather/active?page=0&size=50 USER Paginated active weather alerts from Elasticsearch read model
GET /api/weather/location?latitude={lat}&longitude={lon} USER NOAA active alerts near coordinates
GET /api/weather/state/{stateCode} USER NOAA active alerts by US state code
GET /api/weather/conditions/current?latitude={lat}&longitude={lon} USER Latest NOAA observations for a coordinate
GET /api/weather/conditions/forecast?latitude={lat}&longitude={lon}&hours=48 USER NOAA hourly forecast conditions (1..168h window)
GET /api/weather/search/location/{location} USER Search indexed weather by location text
GET /api/weather/search/event/{eventType} USER Search indexed weather by event type

Criteria query filters

GET /api/criteria/user/{userId}?temperatureUnit=F&monitorForecast=true&hasRainRule=true

Supported filters: temperatureUnit, monitorCurrent, monitorForecast, enabled, hasTemperatureRule, hasRainRule.

Common response codes

200 / 204: Request succeeded.

400: Invalid payload or query parameters.

401: Missing, expired, or invalid token.

403: Role does not allow the endpoint.
